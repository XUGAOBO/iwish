# vue-templete
```
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm start

# 打测试环境包
npm run dev

# 打预发环境包
npm run pre

# 打线上包
npm run build
```

## 移动端rem自适应说明

设计稿`px`到`rem`单位的换算率为100，例如：设计稿宽度为300px，`css`的`rem`单位为300/100 = 3rem
> 如果设计稿不是750的，修改adaptive里面的`DESIGN_WIDTH`即可
